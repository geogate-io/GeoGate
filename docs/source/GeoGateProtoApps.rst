******************************
GeoGate Prototype Applications
******************************

The GeoGate prototype applications demonstrate a wide range of features implemented by the using GeoGate
co-processing component. The aim of the example applications are to provide example and template codes 
that are directly applicable to real world situations and might be used as an easy starting point to 
develop new plugins for GeoGate and/or coupled modeling applications using GeoGate generic ESMF/NUOPC
component.
